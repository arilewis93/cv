03Selected Work

Online MacAdmin Toolbox

Flask · Microsoft Graph / Intune · Entra ID SSO · Render · browser-native
Flagship

A single website that gives IT teams 27 tools for setting up and managing company Macs and PCs — all in the browser, behind a secure company login, with nothing to install. Jobs that used to need separate apps, scripts and expert know-how are now point-and-click.

◆ The hard one — Remote Build & Deploy

Normally, preparing the setup package that gets installed on a company's new computers needs a Mac, specialist software and real expertise — a wall most people can't get over. My tool removes the wall: an administrator prepares the setup once and emails a single link. Whoever receives it — on any device (including mobile phones), anywhere — clicks once, and everything builds and installs itself automatically, exactly as approved and locked to that one person. No Mac, no technical skill, no training. A genuinely new answer to a problem everyone else solves with "go buy a Mac."

Remote package build & deploy Microsoft Graph / Intune Entra ID SSO Zero-install Native macOS agent

ProjectGate

Flask · PostgreSQL · MCP server · multi-tenant SaaS · Render

A platform that manages client projects from start to finish — agreeing what will be built, getting it signed off, tracking the work, testing it, and final approval — with secure digital signatures at every step that can't be tampered with and leave a full paper trail. Logins are passwordless and secure. The clever part: the whole system can also be driven safely by an AI assistant, so routine project admin can be handled just by asking — with the AI only ever allowed to do what it's been given permission to.

E-signature + tamper detection MCP server Entra SSO / passkeys AI scoping interview RBAC + audit logging

SwiftSetup SwiftSetup.cloud

Flask · PostgreSQL · Render · MDM-agnostic · OIDC + passkeys

An online service that turns a brand-new Mac into a fully set-up, ready-for-work company computer on its own — the moment it's switched on, with no one having to touch it. It works no matter which management system the customer already uses, bills automatically per computer, lets companies approve which apps staff can install, and builds each company's custom setup for them. I built, billed and ran the whole thing myself.

Signed PKG generation Per-device billing MDM-agnostic MSP multi-tenancy Installomator delivery

swagger2MCP

FastAPI · async PostgreSQL · MCP · multi-tenant · Render

A tool that lets an AI safely operate almost any other software a business already uses. Point it at a description of another system and, in minutes, it builds a secure connection the AI can use — logins handled automatically and each person's credentials kept encrypted. A built-in approval step means every connection is checked before a team relies on it. In short: it teaches AI assistants to safely use the other tools around them.

OpenAPI → MCP Auto auth wiring Encrypted per-user secrets Approval workflow SSRF protection
